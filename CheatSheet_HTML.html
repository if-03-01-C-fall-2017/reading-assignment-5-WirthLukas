Cheat Sheet

HTML

HTML Elemente

Ein HTML-Element besteht normalerweise aus einem Start-Tag und einem End-Tag, mit dem Inhalt dazwischen. 

						<tagname>Inhalt... </tagname>

Void Elements

HTML-Elemente ohne Inhalt werden als void elements bezeichnet. Void elements können im Start-Tag geschlossen werden:
z.B.: <br /> (Zeilenumbruch)

DOCTYPE

Die !DOCTYPE Dekleration ist kein HTML-Tag. Sie ist eine Anweisung für den Webbrowser, über welche HTML Version die Seite geschrieben wird. (In HTML 5 sieht das so aus: <!DOCTYPE html>)

Attribute

  • Alle HTML-Elemente können Attribute haben
  • Attribute liefern zusätzliche Informationen zu einem Element
  • Attribute werden immer im Start-Tag angegeben
  • Attribute kommen normalerweise in Name/Wert Paare vor. Wie: name = "value"

Das href-Attribut

HTML-Links werden mit dem <a>-Tag definiert. Die Linkadresse wird im href-Attribut angegeben.
z.B.: <a href="https://www.w3schools.com">Das ist ein Link</a>

Das scr-Attribut

HTML-Bilder werden mit dem <img>-Tag definiert. Der Dateiname der Bildquelle wird im src-Attribut angegeben.

Das alt-Attribut

Das alt-Attribut gibt einen alternativen Text an, der verwendet werden soll, wenn ein Bild nicht angezeigt werden kann.

Das lang-Attribut

Die Sprache des Dokuments kann im <html>-Tag mit dem lang-Attribut deklariert werden.

Das title-Attribut

Das Titelattribut liefert zusätzliche "Tool-Tipp" -Informationen, die angezeigt werden wenn man mit der Maus über den Absatz(oder der Überschrift, ...) fährt.

Der "html tag"

  • Er teilt dem Browser mit, dass es sich um ein HTML-Dokument handelt.
  • Er repräsentiert den Stamm eines HTML-Dokuments.
  • Er ist der Behälter für alle anderen HTML-Elemente (mit Ausnahme des !DOCTYPE tags).

Hervorhebungen eines Textes

  • Der em tag: macht einen Text hervorgehobener. Wird meist kursiv dargestellt
  • Der strong tag: markiert wichtigen Text. Wird fett gedruckt dargestellt
  • Der b tag: macht den Text fett gedruckt

Die beiden wichtigsten HTML Elemente

Das Head Element

Das Head-Element ist ein Container für Metadaten über das HTML-Dokument. Sie werden nicht angezeigt. Metadaten definieren typischerweise den Dokumententitel, den Zeichensatz, die Stile, die Links, die Skripte und andere Metainformationen.

Das Title Element

Es definiert den Titel des Dokuments und ist in allen HTML/XHTML-Dokumenten erforderlich.

Das Link Element

Es wird verwendet, um mit externen Stylesheets zu verknüpfen.
z.B.: <link rel="stylesheet" type="text/css" href="StyleSheet.css">

Das Meta Element

Mit ihm werden Zeichensätze, Seitenbeschreibung, Schlüsselwörter, Autor und andere Metadaten festgelegt. Beispiele:

				<meta charset="UTF-8">
				<meta name="description" content="Free Web tutorials">
				<meta name="keywords" content="HTML,CSS,XML,JavaScript">
				<meta name="author" content="John Doe">

Das Body Element

Es enthält alle Inhalte eines HTML-Dokuments wie Text, Hyperlinks, Bilder, Tabellen, Listen usw.

Weitere wichtige HTML Elemente

Der Artikel

Das Artikel ELement gibt unabhängige, in sich geschlossene Inhalte an. Ein Artikel sollte selbstverständlich sein und es sollte möglich sein, ihn unabhängig vom Rest der Seite zu verteilen. Syntax: 

						<article>
							content
						</article>

Zeilenumbruch

Verwenden Sie <br/>, wenn Sie einen Zeilenumbruch (eine neue Zeile) wollen, ohne einen neuen Absatz zu starten.

Überschriften

Überschriften sind mit den <h1> bis <h6> tags definiert. <h1> wichtigste Überschrift. <h6> am wenigsten wichtigste Überschrift. z.B.: <h1>Überschrift 1</h1>

<hr/> definiert eine thematische Veränderung des Inhalts. Wird als horizontale Linie dargestellt.

Paragraphen

Das HTML <p>-Element definiert einen Absatz. Syntax: 

						<p>
							Inhalt
						</p>

<pre> definiert vorformatierten Text.

Bilder

In HTML werden Bilder mit dem <img>-Tag definiert. Es enthält nur Attribute und hat keinen Schluss-Tag. Das src-Attribut gibt die URL (Webadresse) des Bildes an. alt-Attribut: siehe Attribute.
Syntax: 

						<img scr="url" alt="irgendein Text"/>

Section

Das <section>-Tag definiert Abschnitte in einem Dokument, wie z. B. Kapitel, Kopfzeilen, Fußzeilen oder andere Abschnitte des Dokuments.
Syntax: 

						<section>
							Inhalt
						</section>

Links

HTML-Links sind Hyperlinks. Man kann auf einen Link klicken und zu einem anderen Dokument springen.
Syntax: 

						<a href="url">link text</a>

Bookmarks: Zuerst muss man ein Bookmark mit dem id-Attribut machen. z.B.: <h1 id="c1">Chapter 1</h1> Dann beim href (href="#")und nach dem # die id eingeben und du hast eine Verlinkung zu der Überschrift.

Kommentare

Kommentar-Tags werden verwendet, um Kommentare in den HTML-Quellcode einzufügen. Syntax: 

						<!-- Write your comments here -->

Listen

Ungeordnete Listen

Eine ungeordnete Liste beginnt mit dem <ul>-Tag. Jedes Listenelement beginnt mit dem <li>-Tag.

Geordnete Liste

Eine geordnete Liste beginnt mit dem <ol>-Tag. Jedes Listenelement beginnt mit dem <li>-Tag.

Beschreibungsliste

Eine Beschreibungsliste ist eine Liste von Begriffen, mit einer Beschreibung jedes Begriffs. Der <dl>-Tag definiert die Beschreibungsliste, der <dt>-Tag definiert den Begriff (name) und der <dd>-Tag beschreibt jeden Begriff. z.B.: 

					<dl>
						<dt>Coffee</dt>
						<dd>- black hot drink</dd>
						<dt>Milk</dt>
						<dd>- white cold drink</dd>
					</dl>

Listen in Listen

Man kann auch Liste in Listen machen.

Tabellen

  • Verwenden Sie das HTML <table> -Element, um eine Tabelle zu definieren
  • Verwenden Sie das HTML <tr> -Element, um eine Tabellenzeile zu definieren
  • Verwenden Sie das HTML <td> -Element zum Definieren von Tabellendaten
  • Verwenden Sie das HTML <th> -Element, um eine Spaltenüberschrift zu definieren
  • Verwenden Sie das HTML <caption> -Element, um eine Tabellenüberschrift zu definieren.

Zeilen und Spalten verbinden

Verwenden Sie das colspan-Attribut, um eine Zelle in viele Spalten zu verteilen. In diesen Fall würde die SPaltenüberschrift über 2 Zellen gehen. 

						<th colspan="2">Telephone</th>

Verwenden Sie das rowspan-Attribut, um eine Zelle über viele Zeilen hinweg zu bilden. In diesen Fall würde Die Zelle über 2 Zeilen gehen 

						<th rowspan="2">Telephone:</th>
